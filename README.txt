Files:
1. code book.md. 
	It includes the description of the variables and the process required to perform the data cleansing.
2. run_analisis.R. 
	Execute the 5 steps indicated in the project instructions.
		1. Merges the training and the test sets to create one data set.
		2. Extracts only the measurements on the mean and standard deviation for each measurement.
		3. Uses descriptive activity names to name the activities in the data set
		4. Appropriately labels the data set with descriptive variable names.
		5. From the data set in step 4, creates a second, independent time, and data set with the average of each variable for each activity and each subject.
3. “final.csv” file that contains the final result of running the program, to run_analisis.R and that is saved in the working directory.
